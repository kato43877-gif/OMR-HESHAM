# مؤسسة الدكتور عمر هشام الخيرية · Omar Hesham Foundation

## نظرة عامة على المشروع — Project Overview
- **الاسم / Name**: مؤسسة الدكتور عمر هشام الخيرية (Omar Hesham Foundation)
- **الهدف / Goal**: موقع إنساني فاخر لمؤسسة خيرية يعرض الحملات، التبرّعات، التطوّع، الإنجازات، والشفافية المالية بتصميم عالمي المستوى.
- **النوع / Type**: تصميم كامل جديد (Complete premium redesign) — هوية بصرية مستلهمة بالكامل من شعار المؤسسة.

## الميزات المكتملة — Completed Features
- ✅ شاشة تحميل فاخرة (Premium preloader) بالشعار

## معمارية البيانات — Data Architecture
- **مصدر المحتوى / Data source**: `src/data.tsx` (محتوى ثابت بالعربية: حملات، إنجازات، أخبار، فعاليات، قصص، أسئلة، إحصائيات، شركاء).
- **النماذج / Models**: Campaign, Achievement, NewsItem, Event, Story, FAQ, Stat, Program, Value.
- **التخزين / Storage**: حاليًا محتوى ثابت (يمكن ربطه لاحقًا بـ Cloudflare D1/KV عند الحاجة لبيانات ديناميكية حقيقية).

## دليل المستخدم — User Guide
1. تصفّح الموقع عبر القائمة العلوية أو قائمة الجوال.
2. للتبرّع: اذهب إلى `/donate`، اختر المبلغ والحملة وأكمل النموذج.
3. للتطوّع: `/volunteers` ثم اختر المجال واملأ الاستمارة.
4. للاطّلاع على الأثر: `/transparency` و`/achievements`.
5. لوحة الإدارة: `/dashboard` (عرض توضيحي).

## التقنيات — Tech Stack
- **Hono** (TypeScript) على **Cloudflare Pages**
- **Vite** للبناء
- **Chart.js** للرسوم البيانية
- **FontAwesome 6** للأيقونات · خطوط **Tajawal / Cairo / Plus Jakarta Sans**
- CSS مخصّص بالكامل (نظام تصميم واحد · لا إطارات ثقيلة)
